# HR Analytics Dashboard

This project is an interactive Power BI dashboard designed to analyze and visualize key HR metrics. It provides insights into employee performance, attrition rates, hiring trends, and department-level analysis. The goal of this project is to help HR professionals make data-driven decisions that improve workforce management and retention strategies.
